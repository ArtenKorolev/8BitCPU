# ROMS
## WozMonAndIntegerBASIC.rom
*Origin* = **0xD000**

### Description
This ROM contains original WozMonitor and Integer BASIC. Firstly it runs WozMon, you can run
Integer BASIC by typing `E000G` in WozMonitor. Integer BASIC cannot perform floating point operations.

## WozMonAndApplesoftBASIC.rom
*Origin* = **0xD000**

### Description
This ROM contains original WozMonitor and Applesoft BASIC. Firstly it run BASIC but you can run WozMon by typing `CALL -151` in BASIC. Applesoft BASIC can perform floating point operations. This ROM is the most 
interesting one because it can do floating point arithmetic and it's prettier.
